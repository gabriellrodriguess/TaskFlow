# 📋 TaskFlow

Aplicação web de gerenciamento de tarefas com interface moderna e sistema de filtros por status. Permite organizar a rotina de forma visual e eficiente, com persistência de dados via `localStorage`.

![Preview](https://gabriellrodriguess.github.io/Portifolio-Gabriel-/taskflow.png)

🔗 **[Ver projeto no ar](https://gabriellrodriguess.github.io/TaskFlow/)**

---

## ✨ Funcionalidades

- ➕ Adicionar novas tarefas com campo de input
- ✔️ Marcar tarefas como concluídas
- 🗑️ Remover tarefas individualmente
- 🔍 Filtrar tarefas por status: **Todas · Pendentes · Concluídas**
- 💾 Persistência de dados com `localStorage`
- 📱 Layout responsivo e design limpo

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica da página |
| CSS3 | Estilização, layout e responsividade |
| JavaScript (ES6+) | Lógica, filtros e manipulação do DOM |
| LocalStorage | Persistência dos dados no navegador |

---

## 💡 O que aprendi neste projeto

- Como implementar um sistema de **filtros dinâmicos** com JavaScript puro
- Gerenciar estado da aplicação (qual filtro está ativo) sem frameworks
- Renderizar listas condicionalmente com base em propriedades dos objetos
- Persistir e recuperar arrays de objetos com `JSON.stringify` e `JSON.parse`
- Construir interfaces mais elaboradas com foco em UX
